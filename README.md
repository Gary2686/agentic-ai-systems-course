# Building Agentic AI Systems Course Companion

Interactive bilingual course companion for teaching agentic AI systems.

## What is included

- Chapter-based learning interface for 11 chapters
- Traditional Chinese and English language toggle
- Concept maps, definitions, learning steps, examples, and takeaways
- Randomized single-choice quiz for each chapter with instant scoring
- Static frontend structure that can later connect to a database or CMS
